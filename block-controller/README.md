# BlockChain Controller project with Node.js

This project demonstrates the creation of a web service that respond with the block hash.

## Steps to follow

1. Clone the repository to your local computer.
2. Open the terminal and install the packages: `npm install`.
3. Open the file `app.js`and `BlockController.js` and start coding.
4. Run your application `node app.js`
5. Test your Endpoints with Curl or Postman.
